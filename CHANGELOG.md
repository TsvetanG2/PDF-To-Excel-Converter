## [0.2.0-demo] - 2025-06-28

### Added

- Web application (Flask) to convert PDF documents into Excel files.
- "All Text + Tables" mode: extracts all text and tabular data into a single
  Excel worksheet.
- "Tables Only" mode: extracts each detected table into a separate sheet.
- Web interface for uploading PDFs and selecting the extraction mode.
- Desktop application variant.
- Documentation in the `docs/` folder.
